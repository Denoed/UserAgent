
## UserAgents

*A module providing access* <br>
*to the latest **UserAgents**.*

<br>
<br>

## Example

*Returning the agents listed at  [`Common.json`][Common]*

```js
import { common } from 'UserAgents'

const agents = await common();

console.log('User Agents',agents);
```
